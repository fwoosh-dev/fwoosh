## Installation

To use this plugin first install the package:

```sh
npm i --save-dev @fwoosh/tool-{{kebab}}`
# or
yarn add -D @fwoosh/tool-{{kebab}}`
```

Then add it to your `fwoosh.config.ts`:

```ts
export const config: FwooshConfig = {
  plugins: ["@fwoosh/tool-{{kebab}}`"],
};
```
