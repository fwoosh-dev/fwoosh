# Plugins

Plugins are a way to extend the functionality of your fwoosh application.

There are four types of plugins:

- Renderer - Add the ability to render stories for a specific framework
- Panel - Add a tool panel to the `/workbench` view
- Tool - Add a toolbar tool to the `/workbench` view
- Decorators - Common wrappers for stories (<Link to="Features/Decorators">Read more</Link>)

## Configuration

Plugins are configured in the `fwoosh.config.ts` file.

```ts fwoosh.config.ts focus=2:7
export const config: FwooshOptions = {
  plugins: [
    // Just list the name of the plugin to use it
    "@fwoosh/panel-story-description",
    // Or provide an array with the name and options
    ["@fwoosh/panel-story-description", { title: "Comment" }],
  ],
};
```

You can also import the plugins directly and use them in your config.
This provides a better auto-complete experience in your editor.

```ts focus=1,5:7
import StoryDescriptionPanel from "@fwoosh/panel-story-description";

export const config: FwooshOptions = {
  plugins: [
    new StoryDescriptionPanel({
      title: "Comment",
    }),
  ],
};
```

## Parameters

Plugins can also be configured using parameters.

Parameters can be configured in a story file's `meta` object.
This will set the parameter for all stories in the file.

```tsx Example.stories.tsx focus=2:4
export const meta: StoryMeta = {
  parameters: {
    PLUGIN: "OPTION",
  },
};
```

or directly on a story.
This will set the parameter for that specific story.

```tsx Example.stories.tsx focus=3:5
export const BasicStory = () => {};

BasicStory.parameters = {
  PLUGIN: "OPTION",
};
```

### Disable a plugin

To disable a plugin, set the parameter to `false`.

```tsx Example.stories.tsx focus=3:5
export const BasicStory = () => {};

BasicStory.parameters = {
  PLUGIN: false,
};
```
