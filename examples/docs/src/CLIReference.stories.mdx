# CLI Reference

```sh
Synopsis

  $ fwoosh <command> <options>

Commands

  dev     Start the development server
  build   Do a production build of the website
  serve   Run a server a production build of the website
  clean   Clean up all the output files

Global Options

  -h, --help    Display the help output
```

## `dev`

Start the development server

```sh
Options

  --log-level log | info | debug | trace | perf   The amount of logs to print
  --perf                                          Measure performance of fwoosh and plugins.
  --stories string[]                              Globs to match story files
  --open workbench | docs                         Open the browser to the dev server
  -h, --help                                      Display the help output
```

## `build`

Do a production build of the website.

```sh
Options

  --log-level log | info | debug | trace | perf   The amount of logs to print
  --perf                                          Measure performance of fwoosh and plugins.
  --stories string[]                              Globs to match story files
  --out-dir string                                The directory that the built website should
  -h, --help                                      Display the help output
```

## `serve`

Run a server for the production build of the website.

```sh
Options

  --open workbench | docs   Open the browser to the dev server
  -h, --help                Display the help output
```

## `clean`

Clean up all the output files

```sh
Options

  --log-level log | info | debug | trace | perf   The amount of logs to print
  --perf                                          Measure performance of fwoosh and plugins.
  -h, --help                                      Display the help output
```
