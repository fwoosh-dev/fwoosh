# CLI Reference

```sh
Synopsis

  $ fwoosh <command> <options>

Commands

  build   Do a production build of the website
  dev     Start the development server
  clean   Clean up all the output files

Global Options

  -h, --help    Display the help output
```

## `build`

Do a production build of the website.

```sh
Options

  --stories string[]                       Globs to match story files
  --out-dir string                         The directory that the built
                                           website should
  --log-level log | info | debug | trace   The amount of logs to print
  -h, --help                               Display the help output
```

## `dev`

Start the development server

```sh
Options

  --stories string[]                       Globs to match story files
  --log-level log | info | debug | trace   The amount of logs to print
  --open storybook | docs                  Open the browser to the dev
                                           server
  -h, --help                               Display the help output
```

## `clean`

Clean up all the output files

```sh
Options

  --log-level log | info | debug | trace   The amount of logs to print
  -h, --help                               Display the help output
```
